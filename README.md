# CryptoTradeSimulator

## The application allows the user to simulate investing in selected cryptocurrencies
  1. Firebase
   	 - Database
	 - Authentication
  2. CocoaPods
	  - Charts
	  - pod CoinpaprikaAPI
	  - TinyConstraints
	  - Firebase/Auth
	  - Firebase/Database
	  - Firebase/Firestore
	  - SCLAlertView
	  - IQKeyboardManagerSwift
  3. Design Patterns
  	  - MVC
	  - Delegate Pattern
	  - MVVM
  4. JSON
  5. Auto Layout
  6. Apple Pay
